Управляем скоростью доступа к интернет в Ubuntu
Автор: Максим Мишутин | 01.02.2012 | 4:16 | В рубриках: Интернет, Новости, Программы

Программа Trickle позволяет ограничить скорость доступа программ к Интернет каналу без необходимости каких либо настроек firewall’a либо прав супер-пользователся в системе. В режиме сервиса trickle лимитирует скорость использования канала большим количеством пользовательских приложений. В режиме одиночного приложения trickle будет уменьшать скорость доступа к каналу для приложения, которое вы ему явно укажете.
Устанавливается Trickle командой:

Консоль

:~$ sudo apt-get install trickle


Запускаем программу командой:
Консоль

:~$ trickle -d ∞ -u ∞ command


Здесь ∞ – скорость загрузки и исходящая скорость в Kb/s, а command – имя программы, которой устанавливается ограничение скорости.
Примеры использования:
Ограничим скорость загрузки обновлений дистрибутива 500 Kb/s.
Консоль

:~$ trickle -d 500 apt-get upgrade


Теперь ограничим скорость браузеру firefox, он будет загружать страницы со скоростью 100 Kb/s и отправлять данные на скорости 65 Kb/s.
Консоль

:~$ trickle -d 100 -u 65 firefox


Установим лимит скорости в 300 Kb/s на скачивание файла.
Консоль

:~$ trickle -d 300 wget http://cdimage.ubuntu.com/daily/20120201/precise-alternate-i386.iso


Запустив trickle в режиме сервиса, можно ограничить скорость для всех пользователей и все программы. Например, ограничим скорость загрузки в 500 Kb/s, а исходящую скорость ограничим 100 Kb/s.
Консоль

:~$ sudo trickled -d 200 -u 100
