total: вся доступная память, за вычетом съеденной ядром.

 

used: память, как-то используемая. Обычно очень близко к total.

 

free: память, никак не испольщуемая. Обычно очень близко к нулю.

 

shared: странная цифра. Кажется, суммарный объем разделяемой памяти

по всем использующим ее программам. Hепонятно что значит.

 

buffers: дисковые буфера (кэш). Используются при I/O помимо VM.

 

cached: VM кэш.

 

Кэш динамический, то есть для выполения программ без свопа доступно

free + buffers + cached, что отражается во второй строке вывода

free.
